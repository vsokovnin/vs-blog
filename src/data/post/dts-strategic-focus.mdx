**Стратегический фокус** — это одно из четырёх измерений бизнес-модели, в котором компания стремится превосходить конкурентов. Например, по данным [McKinsey](https://www.mckinsey.com/capabilities/growth-marketing-and-sales/our-insights/the-new-consumer-decision-journey), компании с чётким фокусом на клиентских сегментах повышают ROI маркетинга до 15%. Но главный вопрос стратегии — не «что делать?», а «чего НЕ делать?»

## Что такое стратегический фокус

Стратегический фокус — это ключевой этап методологии **дизайн-мышления для стратегии (Design Thinking for Strategy, DTS)**. Он завершает фундаментальный уровень процесса и определяет направление для разработки детализированной бизнес-модели.

Как писал Майкл Портер в [классической статье HBR](https://hbr.org/1996/11/what-is-strategy): «Стратегия — это выбор того, чего НЕ делать». Компании, которые пытаются быть лучшими во всём, в итоге не превосходят конкурентов ни в чём.

### Формула стратегии в DTS

```
СТРАТЕГИЯ = Стратегический фокус + Бизнес-модель + Конкурентное позиционирование
```

Стратегический фокус — первый компонент формулы. Он определяет измерение, с помощью которого компания будет конкурировать и дифференцироваться на рынке.

### Место в процессе DTS

```
Б:Бриф → С:Среда ──┬──→ Ф:ФОКУС → Н:Наблюдение → И:Изучение → П:Проектирование → ...
                   │
           Т:Тренды┘
```

Процесс выбора стратегического фокуса (Ф) запускается после завершения [анализа среды](/blog/dts-environment-analysis) и [анализа трендов](/blog/dts-trend-analysis). Результаты обоих анализов синтезируются для обоснованного выбора фокуса.

**Контрольная точка 1:** Выбор стратегического фокуса — это первая контрольная точка, когда ЛПР (Decision Takers) утверждают выбранное направление перед переходом к детализации бизнес-модели.

## Четыре типа стратегического фокуса

Методология DTS выделяет четыре измерения упрощённой бизнес-модели, каждое из которых может стать стратегическим фокусом.

```
┌─────────────────────────────────────────────────────────────────┐
│  ДИФФЕРЕНЦИАЦИЯ                                                 │
├─────────────────┬─────────────────┬─────────────────────────────┤
│   КЛИЕНТЫ       │   ПРЕДЛОЖЕНИЯ   │      КОМПЕТЕНЦИИ            │
│                 │                 │                             │
│ - Персоны       │ - Ценностное    │ - Деятельности              │
│ - Jobs-to-be-   │   предложение   │ - Ресурсы                   │
│   done          │ - Продукты/     │ - Навыки                    │
│ - Взаимоотно-   │   услуги        │                             │
│   шения         │                 │                             │
├─────────────────┴─────────────────┴─────────────────────────────┤
│                          ФИНАНСЫ                                │
│        - Структура затрат    - Потоки доходов                   │
├─────────────────────────────────────────────────────────────────┤
│  ЛИДЕРСТВО ПО ИЗДЕРЖКАМ                                         │
└─────────────────────────────────────────────────────────────────┘
```

**Ключевой принцип:** Успешная компания превосходит в ОДНОМ компоненте и остаётся конкурентоспособной в трёх других.

### Фокус на клиентах

**Характеристики:**
- Понимание jobs-to-be-done лучше конкурентов
- Клиенты на первом месте при любом решении
- Создание ценности для клиентов, а не просто «забота о клиенте»

**Примеры:** Singapore Airlines, Starbucks, Hilti

**Когда выбирать:**
- Компания имеет глубокие отношения с клиентами
- Клиенты готовы платить за персонализацию
- Конкуренты фокусируются на других измерениях

По данным исследования [КиберЛенинки](https://cyberleninka.ru/article/n/evolyutsiya-i-kontseptsii-segmentatsii-rynka), концепции сегментации рынка эволюционировали от простых демографических моделей к поведенческой и психографической сегментации. Современные подходы включают микросегментацию и персонализацию на уровне индивидуального клиента.

### Фокус на предложениях

**Характеристики:**
- Изобретения и инновации в центре стратегии
- Предсказание того, что клиенты будут ценить в будущем
- Создание потребностей, о которых клиенты ещё не думали

**Примеры:** Apple под руководством Steve Jobs, Tesla

**Когда выбирать:**
- Компания способна к радикальным инновациям
- Готовность к высокому риску ради высокого вознаграждения
- Подрывная (disruptive) стратегия

**Риск:** Фокус на предложениях требует постоянного притока инноваций. Если поток иссякает, компания теряет конкурентное преимущество.

### Фокус на компетенциях

**Характеристики:**
- Превосходящие навыки и/или активы
- Использование уникальных ресурсов для конкурентного преимущества
- Часто — стратегия быстрого последователя (fast follower)

**Примеры:** Amazon (управление цепочкой поставок), IKEA (оптимизация производства)

**Когда выбирать:**
- Уникальные компетенции сложны для копирования
- Инкрементальные улучшения предпочтительнее радикальных инноваций
- Ресурсный подход к стратегии (resource-based view)

### Фокус на финансах

**Характеристики:**
- Лидерство по издержкам
- Инновационные модели ценообразования
- Freemium и платформенные стратегии

**Примеры:** EasyJet, Ryanair, Xiaomi

**Когда выбирать:**
- Ценовая чувствительность высока в отрасли
- Возможность значительно снизить затраты
- Подрывная (disruptive) стратегия через цену

**Важно:** Фокус на финансах — это не просто «дешевле конкурентов». Это системный подход к структуре затрат и потоков доходов, который создаёт устойчивое преимущество.

## Процесс выбора стратегического фокуса

Выбор стратегического фокуса проходит через четыре шага, которые повторяют логику дизайн-мышления: наблюдение → изучение → проектирование → валидация.

```
Ф:1 ─── Наблюдение и изучение
        │
        └── Идентификация преобладающих стратегических
            фокусов в отрасли

Ф:2 ─── Проектирование
        │
        └── Прототипирование возможных стратегических
            фокусов для компании

Ф:3 ─── Валидация
        │
        └── Тестирование гипотез, лежащих в основе
            прототипов

Ф:4 ─── Выбор
        │
        └── Выбор целевого стратегического фокуса
```

### Шаг Ф:1 — Анализ фокусов в отрасли

На этом шаге команда анализирует, какие стратегические фокусы преобладают у конкурентов. Цель — понять «правила игры» в отрасли и найти возможности для дифференциации.

**Для каждого значимого конкурента определить:**

| Элемент | Описание |
|---------|----------|
| **Персона** | Название типа конкурента |
| **Стратегический фокус** | Клиенты / Предложения / Компетенции / Финансы |
| **Тип конкуренции** | Быть превосходящим или быть отличным |
| **3 ключевые характеристики** | Что делает этот фокус уникальным |

**Пример для банковской отрасли:**

| Конкурент | Фокус | Тип | Характеристики |
|-----------|-------|-----|----------------|
| Сбер | Компетенции | Лучше | IT-платформа, экосистема, данные |
| Т Банк | Клиенты | Другой | Digital-first, UX, отсутствие отделений |
| Альфа | Предложения | Лучше | Премиальные продукты, инновации |
| ВТБ | Финансы | Лучше | Масштаб, господдержка, низкие ставки |

### Шаг Ф:2 — Прототипирование фокусов

На этом шаге команда разрабатывает 2-3 прототипа возможного стратегического фокуса для своей компании. Прототип — это гипотеза о том, в каком измерении компания может превосходить.

**Для каждого прототипа описать:**

1. **Выбранный фокус** — одно из четырёх измерений
2. **Обоснование** — почему этот фокус релевантен (ссылка на анализ среды и трендов)
3. **Требуемые изменения** — что нужно изменить в текущей бизнес-модели
4. **Ключевые допущения** — гипотезы, которые должны подтвердиться

**Пример прототипа:**

```
ПРОТОТИП: Фокус на компетенциях (AI-автоматизация)

Обоснование:
- Тренд: рост AI-технологий в отрасли (из Т:3)
- Среда: конкуренты инвестируют в AI медленно (из С)
- Бриф: цель — повышение операционной эффективности

Требуемые изменения:
- Инвестиции в AI-инфраструктуру
- Найм ML-специалистов
- Автоматизация ключевых процессов

Допущения:
- AI-технологии достаточно зрелые для нашей отрасли
- Команда способна освоить новые компетенции за 18 месяцев
- ROI от автоматизации превысит 200% за 3 года
```

### Шаг Ф:3 — Валидация гипотез

Для каждого прототипа проводится быстрая валидация ключевых допущений. Цель — не доказать правильность выбора, а найти опровержения.

**Процесс валидации:**

1. **Сформулировать гипотезы** — явные допущения, которые можно проверить
2. **Разработать эксперименты** — быстрые, дешёвые, с понятными метриками
3. **Провести эксперименты** — искать опровержения, а не подтверждения
4. **Принять решение:**
   - Сохранить прототип (гипотезы подтвердились)
   - Отбросить прототип (критические гипотезы опровергнуты)
   - Изменить и повторить (частичное подтверждение)

**Типы экспериментов:**

| Эксперимент | Что проверяет | Длительность |
|-------------|---------------|--------------|
| Экспертные интервью | Реалистичность допущений | 1-2 недели |
| Анализ аналогов | Успешность подобных стратегий | 1 неделя |
| Пилотный проект | Выполнимость изменений | 2-4 недели |
| Финансовое моделирование | Экономическая жизнеспособность | 1 неделя |

### Шаг Ф:4 — Выбор фокуса

После валидации команда выбирает один стратегический фокус и выносит его на утверждение ЛПР.

**Критерии выбора:**
- [ ] Соответствует руководящим принципам из [стратегического брифа](/blog/dts-strategic-brief)
- [ ] Реалистичен для текущих и развиваемых компетенций компании
- [ ] Отличается от преобладающих в отрасли (для дифференциации) ИЛИ позволяет быть лучше в том же измерении
- [ ] Ключевые гипотезы валидированы
- [ ] Понятен путь к реализации

## Контрольная точка 1: Утверждение стратегического фокуса

После выбора стратегического фокуса настает первая Контрольная точка 1 (КТ / Checkpoint) — формальный момент принятия решения.

**Кто участвует:**
- **ЛПР (Decision Takers):** члены совета директоров, правления, исполнительного комитета
- **Разработчики стратегии (Strategy Designers):** стратегическая команда

**Что утверждается:**
- Выбранный стратегический фокус
- Обоснование выбора (связь с анализом среды и трендов)
- Ключевые допущения и результаты валидации

**Критерии прохождения:**
- Фокус обоснован результатами процессов С и Т
- Фокус соответствует руководящим принципам из процесса Б
- ЛПР готовы взять на себя ответственность за выбранное направление

**Если контрольная точка НЕ пройдена:**
- Вернуться к Ф:2 для пересмотра прототипов
- Или к С/Т для дополнительного анализа

## Типичные ошибки при выборе фокуса

### Ошибка 1: Попытка быть лучшими во всём

**Проблема:** Команда выбирает несколько фокусов одновременно: «Мы будем лучшими в клиентском сервисе И самыми инновационными И самыми дешёвыми».

**Почему это не работает:** Ресурсы компании ограничены. Попытка превзойти во всём приводит к посредственности во всём.

**Решение:** Выбрать ОДИН фокус. Помнить принцип Портера: «Стратегия — это выбор того, чего НЕ делать».

### Ошибка 2: Выбор фокуса без связи с анализом

**Проблема:** Фокус выбирается интуитивно или по предпочтению руководителя, без опоры на анализ среды и трендов.

**Почему это не работает:** Выбранный фокус может не соответствовать реалиям рынка или трендам развития отрасли.

**Решение:** Для каждого прототипа фокуса требовать явную связь с результатами процессов С и Т.

### Ошибка 3: Игнорирование текущих компетенций

**Проблема:** Выбирается привлекательный фокус, который требует компетенций, которых у компании нет и которые сложно развить.

**Почему это не работает:** Стратегия остаётся на бумаге — компания не способна реализовать выбранный фокус.

**Решение:** Оценивать разрыв между текущими и требуемыми компетенциями. Если разрыв критичен — включить план развития компетенций в стратегию.

### Ошибка 4: Копирование фокуса лидера

**Проблема:** Компания выбирает тот же фокус, что и рыночный лидер, надеясь «догнать и перегнать».

**Почему это не работает:** Лидер имеет преимущество масштаба, опыта и ресурсов. Догоняющая стратегия редко приводит к успеху.

**Решение:** Искать незанятые фокусы или фокусироваться на нишевых сегментах, где лидер уязвим.

### Ошибка 5: Отсутствие валидации

**Проблема:** Фокус выбирается на основе обсуждений, без проверки ключевых допущений.

**Почему это не работает:** Критические допущения могут оказаться ложными, и вся стратегия рухнет.

**Решение:** Для каждого прототипа проводить быстрые эксперименты. Искать опровержения, а не подтверждения.

## Чеклист качества выбора стратегического фокуса

**Результаты процесса Ф:**
- [ ] 3-5 преобладающих стратегических фокусов в отрасли описаны
- [ ] 2-3 прототипа стратегического фокуса разработаны
- [ ] Для каждого прототипа сформулированы ключевые гипотезы
- [ ] Эксперименты по валидации проведены
- [ ] Один стратегический фокус выбран и обоснован
- [ ] Контрольная точка 1 пройдена — фокус утверждён ЛПР

**Критерии готовности к переходу на следующий этап:**
- [ ] Фокус чётко определён (одно из четырёх измерений)
- [ ] Связь фокуса с анализом среды и трендов задокументирована
- [ ] Команда понимает, почему выбран именно этот фокус
- [ ] ЛПР подтвердили готовность двигаться в выбранном направлении

## Часто задаваемые вопросы

### Можно ли иметь два стратегических фокуса?

Нет. Стратегический фокус по определению — это ОДНО измерение, в котором компания превосходит. Попытка иметь два фокуса размывает ресурсы и приводит к посредственности в обоих. Однако компания может иметь разные фокусы для разных бизнес-единиц или продуктовых линеек.

### Как часто нужно пересматривать стратегический фокус?

Стратегический фокус — это долгосрочное решение. Типичный горизонт — 3-5 лет. Пересмотр нужен при значимых изменениях: disruption (подрывные инновации) в отрасли, смена технологической парадигмы, M&A. Ежегодный пересмотр — признак отсутствия стратегии.

### Что делать, если ЛПР не могут договориться о фокусе?

Это сигнал о более глубокой проблеме — отсутствии согласия о направлении развития компании. Вернитесь к [стратегическому брифу](/blog/dts-strategic-brief) и уточните руководящие принципы. Если разногласия сохраняются, привлеките внешнего фасилитатора для модерации дискуссии.

### Как выбрать фокус для стартапа без истории?

Для стартапа выбор фокуса определяется гипотезой о конкурентном преимуществе. Проведите customer development, чтобы понять, какое измерение наиболее ценно для целевых клиентов. Стартапы часто начинают с фокуса на предложениях (инновационный продукт) или на клиентах (глубокое понимание ниши).

## Выводы

- **Стратегический фокус — это выбор ОДНОГО измерения** для превосходства: клиенты, предложения, компетенции или финансы. Попытка быть лучшими во всём ведёт к посредственности.

- **Выбор фокуса опирается на анализ.** Сначала изучите преобладающие фокусы в отрасли, затем спроектируйте 2-3 прототипа, валидируйте гипотезы и только после этого выбирайте.

- **Контрольная точка 1 — формальное утверждение.** ЛПР должны принять ответственность за выбранное направление до перехода к детализации бизнес-модели.
